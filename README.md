# PDF Manipulator title generartor
Little web site which generates a fake PDF file "PDF manipulator title" with your name and ID.
This is a little assignment for a subject called Multimedia Applications development.

Pequeño sitio web que genera un pdf falso con el título de "manipulador de PDF con el DNI y el nombre introducido.
Realizado como práctica de la asignatura para la asignatura llamada desarrollo de aplicaciones multimedia.
